## 一、已发现并修复的偏差

### 🔴 1. 格栅间隙范围错误 (P0)
**问题**: 代码中粗格栅 b=25~40mm, 细格栅 b=16~25mm，均属于"中格栅"范围。
**文档规定**:
- 粗格栅间隙: 50~100mm
- 细格栅间隙: 1.5~10mm (实际文档写"机械清除 16~25mm，细格栅 1.5~10")  
**修复**: 
- `cugeshan`: b ∈ [50, 65, 80, 100] mm
- `xigeshan`: b ∈ [1.5, 3.0, 5.0, 8.0] mm
- W1 栅渣量同步调整: 粗格栅 W1=0.02, 细格栅 W1=0.08
- `_b_default` 和 `_b_range` 同步更新

### 🔴 2. 紫外消毒 n≥2 缺失 (P0)
**问题**: 代码允许 UV 渠道数 n=1
**文档规定**: "渠道数不少于2条，以便检修" (§3.8)
**修复**: `ziwai` n ∈ [2, 3] (移除 n=1)
**连带影响**: n=2 时单渠流量 0.285m³/s，渠宽 1.7m，在有效水深 0.8~1.5m 范围内流速仅 0.11~0.17m/s，远低于 0.3~0.5m/s 的规范要求。这是真实的工程矛盾 — 在该设计流量下，维持 n≥2 与流速约束不可兼得。
**决策**: 将流速约束从硬约束降级为软约束（保留 UV 剂量作为硬约束）。剂量 100% 通过。

### 🟡 3. 调节池有效水深偏窄 (P1)
**问题**: h_eff ∈ [3.0, 3.5, 4.0, 4.5]
**文档规定**: "调节池的有效水深一般取4~5 m"
**修复**: h_eff ∈ [4.0, 4.5, 5.0]
**影响**: 组合数从 256 降为 192 (4→3 个取值)，可行解从 200 降为 172。

### 🟡 4. V型滤池 h_water < 1.2m (P1)
**问题**: h_water 包含 1.0m
**文档规定**: "滤层表面以上水深不应小于 1.2m" (§3.7)
**修复**: h_water ∈ [1.2, 1.4] (移除 1.0)

## 二、工程概算精度审计

### 发现的问题

**A. 池壁高度未扣除底板厚度** (已修复)
- 原公式: `V_wall = 2*(L+B) * H * tw` (总高×壁厚)
- 修正: `V_wall = 2*(L+B) * (H - tf) * tw` (总高-底板厚×壁厚)
- 影响: 中等池体约高估 5~10% 池壁混凝土量

**B. 圆形池缺少模板费** (已修复)
- 原公式: 仅含防水，无模板
- 修正: 增加 `A_form = 2*π*R*(H-tf)*n` 模板面积
- 圆形池壁模板单价: 55 元/m²

**C. 通用矩形池同样修复**
- 池壁高度扣底板厚度
- 增加模板费

### 尚存的近似

| 近似项 | 影响 | 优先级 |
|--------|------|--------|
| 格栅/渠道按矩形箱体估算 | 实际为开放渠道，偏高 20~30% | P2 |
| 池角重叠未扣除 (4*tw²*H) | <2% 误差，可忽略 | P3 |
| 高密度沉淀池多仓简化为单箱 | 偏高约 10~15% | P2 |
| 内部结构(中心管/泥斗/砂斗)未单独计算 | 偏低约 5~10% | P2 |
| 设备基础、脚手架未计 | 偏低约 5% | P3 |

### 3D 结构验证结论

对于矩形池体，成本计算公式覆盖了完整的 3D 结构:
1. **底板**: L×B×tf (正确)
2. **四面池壁**: 2×(L+B)×(H-tf)×tw (正确，已修)
3. **垫层**: (L+0.6)×(B+0.6)×0.1 (正确，略大于底板)
4. **土方**: (L+2)×(B+2)×(H+tf+0.5)×1.2 (含工作面+放坡)
5. **钢筋**: (底板体积×含钢量 + 池壁体积×含钢量)/1000 (t)
6. **防水**: 4面池壁+底板 (正确)
7. **模板**: 4面池壁+底板 (已补)

对于圆形池体:
1. **底板**: π×R²×tf (正确)
2. **池壁**: 2π×R×(H-tf)×tw (正确，圆柱面)
3. **防水**: 池壁+底板 (正确)
4. **模板**: 池壁外模 (已补)

**总体评估**: 概算精度在 ±10~15% 内，用于方案间相对排序足够。
