## 三、社区模组新增 (2 个)

### 污水提升泵房 (wuni_tisheng)
```
mods/community/wuni_tisheng/
├── mod.json              # primary, 4参数 (n_work, H_st, v_suction, v_discharge)
├── __init__.py
├── models/__init__.py
├── models/wuni_tisheng.py (218行)
└── discretization.json   # 192 组合, estimator_type: "rectangular"
```

| 设计要素 | 核心公式 |
|---------|---------|
| 单泵流量 | Q_single = Q_design / n_work |
| 管径 | D = √(4Q/πv) |
| 水头损失 | 曼宁公式: i = (n×v/R^(2/3))², R=D/4, n=0.013 |
| 总扬程 | H = H_st + 1.2×(h_f_suction + h_f_discharge) + 1.5 |
| 轴功率 | P = ρgQH/η, η=0.75 |
| 集水池 | V ≥ Q_single × 5min (GB50014 §6.3.1) |
| 备用泵 | n_work≤4→1台, ≥5→2台 (§6.4.1) |

**方案空间**: 192 组合 (4×4×4×3), 约束: v_suction∈[0.7,1.5], v_discharge∈[0.8,2.5], H∈[5,30], V≥V_min

### 巴氏计量槽 (bashi_jiliangcao)
```
mods/community/bashi_jiliangcao/
├── mod.json              # tertiary, 1参数 (b: 喉宽)
├── __init__.py
├── models/__init__.py
├── models/bashi_jiliangcao.py (182行)
└── discretization.json   # 5 标准喉宽, estimator_type: ""
```

| 设计要素 | 核心公式 |
|---------|---------|
| 流量公式 | Q = C × h_a^n (CJ/T3008.5-93) |
| 5种标准喉宽 | 0.152, 0.228, 0.30, 0.45, 0.60 m |
| 渠道宽度 | B = b × 3 (喉宽≈1/3渠道宽) |
| 淹没度 | S = h_b/h_a ≤ 0.6 (自由出流) |
| 弗劳德数 | Fr = v/√(gh_a) < 1.0 |
| 上下游直段 | L_up≥10B, L_down≥5B |

**方案空间**: 5 组合 (5种喉宽), 约束: 标准喉宽, Fr<1, v∈[0.3,2.0]
